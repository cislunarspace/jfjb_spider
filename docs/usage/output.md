# 输出说明

## 目录结构

### 解放军报

```
output/
  2026-03-10/
    第01版_要闻/
      01_文章标题一.pdf
      02_文章标题二.pdf
    第02版_军事新闻/
      01_文章标题三.pdf
    解放军报_2026-03-10_全集.pdf
```

### 人民日报

```
output/rmrb/
  2026-03-10/
    第01版_要闻/
      01_文章标题一.pdf
    人民日报_2026-03-10_全集.pdf
```

## 输出模式

| 模式 | 参数 | 生成内容 |
|------|------|----------|
| 默认 | 无 | 单篇 PDF + 合集 PDF |
| 仅合集 | `--combined-only` | 只生成一个带书签目录的合集 PDF |
| 仅单篇 | `--individual-only` | 每篇文章一个 PDF，按版面分目录 |

## PDF 内容

每个 PDF 包含：

- **标题** — 黑体（SimHei），20pt，居中
- **副标题** — 黑体，11.5pt，居中（如有）
- **元信息** — 版面编号、版面名称、来源链接
- **正文** — 宋体（SimSun），11.5pt，首行缩进 24pt
- **英文字符** — 自动切换为 Times New Roman

## PDF 书签

合集 PDF 自动生成两级书签目录：

```
├── 第01版 要闻                    ← 版面书签（顶层）
│   ├── 文章标题一                  ← 文章书签
│   └── 文章标题二
├── 第02版 军事新闻
│   └── 文章标题三
```

在 PDF 阅读器中打开侧边栏即可看到书签导航。

## 文件名规则

文件名中的特殊字符（`\ / : * ? " < > |`）会被替换为下划线，超过 180 字符的文件名会被截断。
