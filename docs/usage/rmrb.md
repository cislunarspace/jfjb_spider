# 人民日报爬虫

通过 HTML 页面解析抓取人民日报各版面文章。

## 基本用法

```bash
# 抓取今天的报纸
python rmrb.py

# 抓取指定日期
python rmrb.py --date 2026-03-10
```

## 工作流程

人民日报使用 HTML 页面（而非 JSON API），爬虫分三步抓取：

1. **发现版面** — 访问版面导航页，获取所有版面链接
2. **提取文章** — 从每个版面页提取文章链接
3. **解析内容** — 逐个访问文章页，提取标题、作者和正文

## 编码处理

人民日报网站使用 GB2312/GBK 编码，爬虫自动处理：

1. 从 HTML `<meta charset>` 标签检测编码
2. 回退到 `requests` 库的自动编码检测
3. 最终回退到 UTF-8

一般情况下无需手动干预。

## 专用参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--base-url` | 站点根地址 | `https://paper.people.com.cn` |

## 与解放军报的区别

| 特性 | 解放军报 | 人民日报 |
|------|----------|----------|
| 数据格式 | JSON API | HTML 页面 |
| 批量抓取 | 支持 | 不支持 |
| 默认输出目录 | `output/` | `output/rmrb/` |
| 编码 | UTF-8 | GB2312/GBK（自动检测） |
